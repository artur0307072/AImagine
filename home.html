Sobre o sistema:
Este projeto permite gerar imagens automaticamente por IA a partir de descrições textuais.
  • Desenvolvido em PHP com Programação Orientada a Objetos (POO).
  • Utiliza APIs como DeepAI, Hugging Face, Leonardo AI ou DeepSeek.
  • Ideal para criadores de conteúdo, designers e profissionais de marketing.
  • As imagens podem ser protegidas por marca d'água personalizada.
  • Encapsulamento, herança, polimorfismo e interfaces são aplicados no backend.
  • Em breve: integração com banco de dados para histórico de imagens.
